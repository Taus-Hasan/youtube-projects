Usage Guide

Ready to run it yourself? Follow these simple steps.

1

Save the Code

Save the Python script as a file named `youtube_downloader.py`.

2

Open Terminal

Navigate to the directory where you saved the file.

3

Run the Script

Execute the command:
`python youtube_downloader.py`

4

Enter URL

When prompted, paste a YouTube video URL and press Enter.
